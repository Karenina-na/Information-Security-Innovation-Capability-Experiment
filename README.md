# 西北工业大学(NWPU)信息安全创新综合实验

## 基于多模态的网络入侵检测系统

### Attack Types

- Brute Force FTP
- Brute Force SSH
- DoS
- Heartbleed
- Web Attack
- Infiltration
- Botnet
- DDoS

### 项目说明

Main 下为运行文件夹，Model.py中有模型和训练代码
Result 存放运行结果以及结果可视化文件
Util 文件夹存放工具文件，包括one-hot编码、标准化函数等
Data 文件夹存放数据集
DataProcess 文件夹存放数据处理代码
